22 Cache then Network strategy

More accurately Cache OR Network but it is know as the above.

If offline and we request index.html the SW looks first in the LOCAL CACHE we created and if it is not there goes to the network.

skipWaiting commented out. We can see no effect when we edit index.html

One can also clear SW by closing all tabs and open new tab, which will reload new SW and new index.html

When ONLINE this will be very fast. When OFFLINE the page can still be rendered if it is in CACHE.
