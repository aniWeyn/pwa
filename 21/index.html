21 PRECACHE SHELL and HOME PAGE

We now precache the APP SHELL (the minimum needed for a page)

This is best done in the INSTALL event.

In pre-caching, if there is any error in the process, no files will load.

MDN Cache API

EX: make a typo in filename. Clear site data, close tab and reload page. See that no files are cached. Then reverse.

There will be an error in CONSOLE.
